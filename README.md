### Цель проекта
Написан в учебных целях

### scripts.py
Программа может:
1. Исправить все плохие оценки (2 и 3) на 5.
2. Удалить все жалобы на ученика.
3. Добавить похвалу по выбранному предмету.
### Подготовка к запуску
1. Скачайте файл с кодом `scripts.py`.
2. Положите его внутрь директории `e-diary-master`
3. Откройте командную строку([инструкция как открыть](https://timeweb.com/ru/docs/obshchie-voprosy/komandnaya-stroka-v-raznyh-os/)), \
пройдите до директории `e-diary-master` (введите команду `cd path`, где `path` полный путь до файла)
4. Введите команду: `python manage.py shell` \
Должно получится похожее:
```commandline
C:\Python\Python_from_0\Devman\Django_orm3\e-diary-master>python manage.py shell
Python 3.9.1 (tags/v3.9.1:1e5d33e, Dec  7 2020, 16:33:24) [MSC v.1928 32 bit (Intel)] on win32
Type "help", "copyright", "credits" or "license" for more information.
(InteractiveConsole)
>>>
```
5. На этом подготовка завершена, дальнейшие шаги будут отличаться в зависимости от целей:
### Подробнее по отдельности:

#### Исправление оценок:
1. Выполнить шаги   'Подготовка к запуску'
2. Введите команду: `from scripts import fix_marks`
```commandline
>>> from scripts import fix_marks
>>>
```
3. Далее введите: `fix_marks(name)`, где вместо `name` строку с ФИО, кому нужно исправить оценки,  например:
```commandline
>>> fix_marks('Русаков Ефрем Жанович')
Оценки исправлены
>>>
```
4. Если все введено правильно, то выведется "Оценки исправлены", если ввести некорректно, то программа подскажет как исправлять
#### Удаление жалоб:

1. Выполнить шаги   'Подготовка к запуску'
2. Введите команду: `from scripts import remove_chastisements`
```commandline
>>> from scripts import remove_chastisements
>>>
```
3. Далее введите: `remove_chastisements(name)`, где вместо `name` строку с ФИО, кому нужно удалить жалобы,  например:
```commandline
>>> remove_chastisements('Русаков Ефрем Жанович')
Жалобы удалены
>>>
```
4. Если все введено правильно, то выведется "Жалобы удалены", если ввести некорректно, то программа подскажет как исправлять
#### Добавление похвал:
1. Выполнить шаги   'Подготовка к запуску'
2. Введите команду: `from scripts import create_commendation`
```commandline
>>> from scripts import create_commendation
>>>
```
3. Далее введите: `create_commendation(name, lesson)`, где вместо `name` строку с ФИО, а вместо `lesson` предмет, по которому добавить похвалу  например:
```commandline
>>> create_commendation('Русаков Ефрем Жанович', 'Математика')
Похвала добавлена
>>>
```
4. Если все введено правильно, то выведется "Похвала добавлена", если ввести некорректно, то программа подскажет как исправлять
